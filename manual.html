概要・特徴

「簡易フローチャート作図ツール(Python版)」は、業務フローや処理ロジックをすばやく図にまとめるためのデスクトップアプリケーションです。Python の Tkinter キャンバスで動作し、インストール不要でローカル環境ですぐに使えます。

主な特徴

  • フローチャートを気軽に作成、気軽に編集可能
  • 縦横のスイムレーン / ６種類の要素 / 要素間の2種類のリンクに対応
  • JSON形式およびMermaid形式ファイルの読み書きに対応
  • 生成AI連携によるフロー自動生成（OpenAI GPT, Google Gemini, Anthropic ClaudeとのAPI接続に対応）
  • NEW ツールで作図したフローチャートデータをJSON形式で保存して、Excel作図データに変換可能
  • NEW ツールで作図したフローチャートデータをMermaid形式で保存して、バイブコーディングの添付資料として活用可能

動作環境

  • Python 実行可能な環境（Windows11 および macOS26.4にて動作確認）
  • Tkinter の Canvas が利用できること（Google Colaboratory など Web 環境は不可）
  • Excelでのフローチャート作図機能は、Windows環境のみ対応（macOS用はベータ版公開中）

    キャンバス

    フローチャートを描画するメイン領域です。スクロールバーでスクロール可能です。グリッドが表示されている場合、要素の配置がグリッドにスナップされます。

    AI パネル（右サイドパネル）

    起動方法

    インストール手順

    前提条件：pythonとgitがインストールされていること。インストールされていない場合は、公式サイトからインストールしてください。
    1. コマンドプロンプト（またはターミナル）を起動し、チェックアウトフォルダに移動します。
    2. 以下のコマンドを実行し、プロジェクトのコードをチェックアウトします。
      git clone https://github.com/YoshiToshi1025/flowchart-drawing-tool-python.git
    3. チェックアウトフォルダで以下を実行して依存パッケージをインストールします。
      pip install -r requirements.txt
    4. 英語版で使いたい場合は constants_en.py を constants.py に上書きコピーします。
    5. AIによるフロー自動生成を行いたい場合は、チェックアウトフォルダ内の .env ファイルに連携したい生成AIのAPI KEYを記述し、設定ファイル(constants.py)内で生成AIモデル名を指定します。

      [.envファイル内]
      OPENAI_API_KEY=(OpenAI GPT接続用API Key)
      GEMINI_API_KEY=(Google Gemini接続用API Key)
      ANTHROPIC_API_KEY=(Anthropic Claude接続用API Key)

      [constants.pyファイル内]
      AI_MODEL=使用する生成AIモデル名
      例) AI_MODEL="gpt-5.4"
    6. flowchart_tool.py を実行します。
    git cloneでチェックアウトしたプロジェクトを最新にするには、git pull を実行します。
    API Key を設定していない場合、生成AI連携機能は利用できませんが、AI 機能以外のフローチャートの手作業での作図は問題なく使用できます。

    実行コマンド

    python flowchart_tool.py

    終了

    ウィンドウを閉じようとすると確認ダイアログが表示されます。自動保存には対応していないので、編集データを保存してから終了するようにしてください。

    Excel作図VBA(Windows版)の使い方

    ExcelファイルにVBAを登録

    1. Excelを起動し、フローチャートを作図したいブックを開く
    2. VisualBasicエディタを起動（VisualBasicエディタの利用方法はネット等でお調べください）
    3. VisualBasicエディタの「ファイル」メニューから「ファイルのインポート」を選択
    4. ファイルダイアログで、 ExcelShapesDrawingWithJSON.bas を選択
    5. 標準モジュールに ExcelShapesDrawingWithJSON が登録されていれば、登録完了です。
    前提条件：VBAはWindows版のExcelで利用可能です。※macOS用のベータ版を公開しています（ファイルパスに日本語が使えません）。
    本VBA実行は単独で動作するため、他のライブラリや参照設定は不要です。

    VBA実行

    1. Excelを起動し、上記VBAが登録されているブックを開く
    2. フローチャートを作図したいシートを選択
    3. マクロ画面で DrawFlowchat を実行
    4. ファイルダイアログで、インポートするJSONファイルを選択
    5. Excelアクティブシートに自動描画されます。必要に応じてフローチャートを調整してください。

    フローチャート要素（ノード）の配置

    配置の手順

    1. ツールバーで配置したい要素のモードボタンを選択します（例: Process）。
    2. キャンバス上で、要素を配置したい位置でクリックします。（マウスボタンを押したまま、位置の微調整ができます）
    3. マウスボタンを離すことで、ノードの位置が確定します。グリッドが ON の場合は、格子点にのみ配置されます。
    4. 要素位置を微調整したい場合、要素をマウスで選択しながらドラッグ操作により位置を調整できます。
    5. 配置したノードをダブルクリックすると、テキスト編集モードになります。（詳細は、基本操作：テキスト編集を参照）

    要素の種類

    Terminator
    始点・終点・サブルーチン
    Process
    処理ボックス（角丸も選択可）
    Decision
    条件分岐（ひし形）
    I/O
    入出力（平行四辺形）
    Storage
    データストレージ（楕円端）
    Document
    書類・ドキュメント
    ツールバーのモードボタンにマウスを合わせると、ツールチップでモード名が表示されます。

    右クリックメニューからの切り替え

    キャンバス上で右クリックすると、ポップアップメニューから各モードへ直接切り替えることもできます。

    スイムレーン

    スイムレーンは、フローチャートの担当者や処理区分をレーン（帯）で視覚的に区切るための要素です。

    配置の手順

    1. ツールバーの「Swimlane」ボタンをクリックします。
    2. キャンバス上でクリックすると、スイムレーンが配置されます。
    3. スイムレーンのヘッダー部分をダブルクリックするとタイトルを編集できます。

    スイムレーンの種類

    種類説明
    横型（Horizontal） レーンが上下に並ぶ。デフォルト設定。担当者ごとの行を横に広げる形式。
    縦型（Vertical） レーンが左右に並ぶ。フェーズや工程を列方向に区切る形式。
    スイムレーンの種類（横型 / 縦型）は、ツールバーのスイムレーンアイコンを右クリックすることで切り替え可能です。

    位置とサイズの変更

    スイムレーンのヘッダーまたはフッター部分を選択しながらドラッグ操作を行うことで、スイムレーンの位置を変更できます。

    スイムレーンを選択した状態で、Shiftキー+マウスホイール操作で、スイムレーンの幅を調整できます。

    スイムレーンを選択した状態で、Ctrlキー+マウスホイール操作で、スイムレーンの長さを調整できます。

    接続線（リンク）の描画

    2 つのノードを矢印線で接続します。

    接続の手順

    1. ツールバーで Link_elbow（折れ線）または Link_straight（直線）をクリックします。
    2. 接続元のノードをクリックします（ノードがハイライトされます）。
    3. 接続先のノードをクリックします。
    4. 矢印線が自動的に描画されます。

    リンクの種類

    Link_elbow（折れ線）
    直角に折れ曲がる配線。自動ルーティング対応
    Link_straight（直線）
    直接つなぐ直線リンク

    線種の変更（直線リンクのみ）

    直線リンクを選択した状態で Ctrl + マウスホイールを操作すると、線種をローテーションで変更できます。

    線種説明
    solid（実線）通常の実線（デフォルト）
    dashed（破線）長めの破線
    dotted（点線）細かい点線

    リンクラベルの追加

    リンクをダブルクリックするとテキスト編集モードになり、ラベルを入力できます。Decision ノードから出るリンクには自動的に Yes / No ラベルが付きます。

    矢印の向き

    デフォルトは接続先に向かう矢印（last）です。constants.py の EDGE_PARAMS["arrow_kind"] で変更できます（"last" / "first" / "both" / "none"）。

    ノード移動時の自動再配線

    ノードを移動すると、接続されているリンクは自動的に再配線されます。手動で微調整したい場合はリンク配線の手動調整をご参照ください。

    テキスト編集

    ノード・リンク・スイムレーンのラベルを直接編集できます。

    編集の手順

    1. Select モードで対象の要素（ノード / リンク / スイムレーンヘッダー）をダブルクリックします。
    2. テキスト入力フィールドが表示されます。
    3. テキストを入力します。
    4. Enter または入力フィールド以外をクリックして確定します。Escape でキャンセルします。

    改行の入力

    テキスト内に \n と入力すると、表示時に改行されます。

    例: データを\n確認する と入力すると 2 行で表示されます。

    選択・移動操作

    単一選択

    1. ツールバーの「Select」をクリックして Select モードにします（またはキー操作）。
    2. ノード・リンク・スイムレーンをクリックすると選択状態になります（青色のハイライト）。
    3. Escape または別の空白エリアをクリックすると選択が解除されます。

    複数選択（矩形選択）

    1. Select モードで、要素のない位置からドラッグすると点線の矩形が表示されます。
    2. 矩形内に含まれるすべての要素が選択状態になります。

    全選択

    Ctrl + A ですべての要素を選択します。

    要素の移動

    1. Select モードで移動したいノードをドラッグします。
    2. 複数選択している場合は、選択中のすべての要素が一緒に移動します。
    3. グリッドが ON の場合は移動もグリッドにスナップされます。
    スイムレーン内のノードを移動する場合も同様の操作で行えます。

    削除

    選択中の要素（ノード・リンク・スイムレーン）を削除します。

    削除の方法

    操作説明
    ツールバーの「Delete」ボタンをクリック選択中の要素を削除します
    Delete キー選択中の要素を削除します（キーボードショートカット）
    右クリックメニュー → 「Delete Selected」同上
    ノードを削除すると、そのノードに接続されているリンクも同時に削除されます。削除後は Undo で削除前の状態に戻せます。

    Undo / Redo

    直前の操作を取り消したり、取り消した操作をやり直したりできます。

    操作方法

    操作説明
    ツールバーの「Undo」ボタン直前の操作を取り消す
    ツールバーの「Redo」ボタン取り消した操作をやり直す
    Ctrl + ZUndo（キーボードショートカット）
    Ctrl + YRedo（キーボードショートカット）
    操作履歴はセッション中のみ保持されます。ツールを終了すると履歴はリセットされます。

    グリッド表示

    キャンバスに格子状のグリッドを表示し、要素の配置をグリッド単位でスナップさせます。

    ON / OFF の切り替え

    ツールバーの「Grid」チェックボックスをクリックして ON/OFF を切り替えます。

    状態説明
    ON（デフォルト）グリッド線が表示される。要素の配置・移動がグリッドにスナップされる
    OFFグリッド線が非表示になる。要素は任意の位置に配置・移動できる
    グリッド間隔は constants.py の CANVAS_PARAMS["grid_spacing"] で変更できます（デフォルト: 15px）。

    キャンバスのリサイズ

    大きなフローチャートを作図するためにキャンバスのサイズをツール画面のサイズより大きくしたり、大きくなってしまったサイズを小さくすることができます。

    操作方法

    1. ツールバーの「Resize Canvas」ボタンをクリックします。
    2. リサイズダイアログが表示されます。新しい縦・横のサイズ（ピクセル指定）を入力します。画面と同サイズにするには、Minimum Sizeのチェックボックスを選択してください。
    3. 確定するとキャンバスが指定サイズに変更されます。
    ツール画面を拡大すると、キャンバスサイズも自動で拡大されます。なお、ツール画面を縮めてもキャンバスサイズ自体は自動で縮小されません。
    ツール画面に表示されている範囲よりキャンバスサイズを小さくすることはできません。

    スクロール

    キャンバスが画面より大きくなった場合、キャンバス右側・下側のスクロールバーでスクロールできます。またはマウスホイールで縦スクロール、Shift + マウスホイールで横スクロールが可能です（リンク非選択時）。

    リンク配線の手動調整

    自動配線されたリンクは、マウスホイール操作で細かく調整できます。まずリンクを選択（クリック）してから操作します。

    折れ線リンク（Link_elbow）の調整

    操作効果
    Ctrl + マウスホイール 接続点を巡回変更します（ノードの上下左右のどの辺から接続するかが変わります）
    Shift + マウスホイール 折れ線の回り込み距離を調整します（折れ曲がりの位置が移動します）
    Ctrl + Shift + マウスホイール リンクのラベル位置を調整します

    直線リンク（Link_straight）の調整

    操作効果
    Ctrl + マウスホイール 線種をローテーションで変更します（実線 → 破線 → 点線 → 実線 …）
    これらの操作はリンクを選択している状態でのみ有効です。

    ノード/スイムレーンの色変更

    選択中のノードの塗りつぶし色をキーボードショートカットで変更できます。

    色変更のショートカット

    キー操作色
    Ctrl + 1Light Pink   #FDECEF
    Ctrl + 2Light Orange   #FFEFF2
    Ctrl + 3Light Yellow   #FFF1E3
    Ctrl + 4Light Yellow2   #FFFDE6
    Ctrl + 5Light Green   #F1FAEE
    Ctrl + 6Light Teal   #EFFFF7
    Ctrl + 7Light Blue   #EAFBFF
    Ctrl + 8Light Indigo   #EEF4FF
    Ctrl + 9Light Purple   #F3EEFF
    Ctrl + 0Light Gray   #F7F5F2
    Ctrl + -白（デフォルト色にリセット）
    複数のノードを選択した状態でショートカットを押すと、選択中のすべてのノードの色が一括変更されます。
    各キーに割り当てる色は、constants.py の NODE_FILL_COLORS を変更することで変えられます。

    ノードのステータス

    ノードに 3 種類のステータスを設定して、表示スタイルを切り替えられます。フローの実行状況や注目箇所を視覚的に示すのに便利です。

    ステータスの種類

    Normal
    Active
    Inactive
    ステータス表示スタイル用途例
    Normal白背景・通常の枠色通常状態（デフォルト）
    Activeレモンイエロー背景・金色の枠・太い枠現在実行中の処理を示す
    Inactive薄グレー背景・グレーの枠・テキスト未使用・無効化した処理を示す

    ステータスの変更方法

    1. ステータスを変更したいノードを選択します（複数選択可）。
    2. 右クリックメニューを開きます。
    3. 「Nodes Status : Normal / Active / Inactive」からいずれかを選択します。

    JSON 保存 / 読み込み

    フローチャートのデータを JSON 形式でファイルに保存・読み込みできます。JSON を使うことで後から再編集が可能です。

    保存

    1. ツールバーの「Save JSON」ボタンをクリックします（または右クリックメニューから）。
    2. ファイル保存ダイアログが開きます。ファイル名と保存場所を指定して「保存」をクリックします。

    読み込み

    1. ツールバーの「Load JSON」ボタンをクリックします。
    2. ファイル選択ダイアログが開きます。読み込む JSON ファイルを選択して「開く」をクリックします。
    3. フローチャートが再描画されます。
    example フォルダにサンプル JSON ファイルが含まれています。Load JSON で読み込んで動作を確認できます。
    読み込み時は現在のキャンバスの内容がクリアされ、上書きされます。必要に応じて、読み込み前に保存するようにしてください。

    画像の保存

    キャンバスの内容を画像ファイルとして書き出します。

    保存手順

    1. ツールバーの「Save Image」ボタンをクリックします（または右クリックメニューから）。
    2. ファイル保存ダイアログが開きます。
    3. ファイル形式（PNG または JPEG）を選択し、ファイル名と保存場所を指定します。
    4. 「保存」をクリックすると画像が保存されます。
    Windows で複数ディスプレイを使用し、ディスプレイごとの拡大率が異なる場合、画像保存が正しく動作しないことがあります。その場合は画面キャプチャツールをご利用ください。

    Mermaid 形式の保存/読み込み

    Mermaid 記法で書かれたフローチャートテキストファイルをファイル保存と読み込みに対応しています。読み込み時は自動的にキャンバスへ配置されます。

    読み込み手順

    1. ツールバーの「Load Mermaid」ボタンをクリックします。
    2. ファイル選択ダイアログが開きます。Mermaid 形式のテキストファイルを選択します。
    3. 自動的にノードとリンクが配置されます（配置は自動調整が必要な場合があります）。

    対応フォーマット例

    ```mermaid
    flowchart TD
      A@{ shape: stadium, label: 開始, bx: 0, by: 0 }
      B@{ shape: rounded, label: "ツールの起動", bx: 0, by: 1 }
      C@{ shape: diamond, label: "新規作成or編集?", bx: 0, by: 2 }
      D@{ shape: rounded, label: "作図", bx: -1, by: 3 }
      E@{ shape: lean-r, label: "データ読込", bx: 1, by: 3 }
      F@{ shape: rounded, label: "編集", bx: 1, by: 4 }
      G@{ shape: lean-r, label: "画像出力", bx: 0, by: 5 }
      H@{ shape: rounded, label: "資料に画像を添付", bx: 0, by: 6 }
      I@{ shape: stadium, label: "終了", bx: 0, by: 7 }

      A --> B --> C -- "新規作成" --> D --> G --> H --> I
      C -- "編集" --> E --> F --> G
    Mermaid形式の保存では、要素の位置、リンク情報は保持されますが、詳細設定内容は保存されません。スイムレーンも保存されません。
    AI 自動生成機能でも同じ Mermaid 形式が出力されます。AI で生成した後に手動で微調整するワークフローでも利用できます。

    フローの自動生成（AI 連携）

    生成AIと連携して、生成AIが自動生成したフローチャートをキャンバスに描画できます。ツールに描画したフローチャートを手動で編集することも可能です。

    事前準備

    チェックアウトフォルダに .env ファイルを作成し、利用したい生成AIサービスの API KEY を記述し、設定ファイル constants.py に使用する生成AIモデルを指定してください。

    [.envファイル内]
    OPENAI_API_KEY=(OpenAI GPT接続用API Key)
    GEMINI_API_KEY=(Google Gemini接続用API Key)
    ANTHROPIC_API_KEY=(Anthropic Claude接続用API Key)
    [constants.pyファイル内]
    AI_MODEL=使用する生成AIモデル名
      例) AI_MODEL="gpt-5.5"
    API Key の取得方法は、利用したい生成AIサービスの公式ドキュメントを参照ください。
    ツールバーに 「AI-generation」ボタンが表示されない場合、生成AI利用のための設定が正しく行われていません。設定内容をチェックしてください。

    操作手順

    1. ツールバーの「AI-generation」アイコンを ON にします（右サイドパネルが開きます）。
    2. 「Flow title:」テキストボックスに生成したい処理フローの概要を、「Specs:」テキストエリアに付け足したい仕様を入力します。
    3. 「Generate」ボタンをクリック（または Enter）します。
    4. AI が Mermaid 形式のフローデータを生成し、生成内容がチャットエリアに表示され、キャンバスに自動生成されたフローチャートが描画されます。
    5. 保存されたMermaidファイルのファイル名が確認ダイアログに表示されます。
    6. 必要に応じて手動で位置・テキスト・リンクを調整します。

    入力例

    商品の注文から出荷までの業務フロー
    ユーザー認証処理のロジック
    AI 自動生成を実行するたびに 生成AIのAPI の利用料が発生します。

    使用モデル

    デフォルトモデル: gpt-5.4（constants.py の AI_MODEL で変更可能）

    制限事項・注意事項

    制限事項

    • キャンバスの拡大・縮小表示（ズーム）には未対応です。
    • Windows で複数ディスプレイを使用し、ディスプレイごとの拡大率が異なる場合、「Save Image」による画像保存が正しく動作しないことがあります。その場合は画面キャプチャツールをご利用ください。
    • Excel作図VBAのmacOS用ベータ版は、(macOS版VBA側の仕様で）ファイルパスおよびファイル名に日本語に対応指定していません。ファイルパスおよびファイル名を半角英数のみにして試してみてください。

    注意事項

    • フローチャートの自動生成には OpenAI/Gemini/Claude の API を利用するため、自動生成を行う毎に各生成モデル側の API利用料 が発生します。
    • Google Colaboratory などの Web 環境では動作しません。
    • macOS では Tkinter が利用できる設定が必要です。
